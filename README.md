# VUPY
A Vagrant command line wrapper.

If you are tired of the need to go the a specific folder to run Vagrant command, vupy is your tool.
Vupy helps you run vagrant command from any folder.

## Installing vupy ##

First, you need to have [Git Bash](https://git-scm.com/download/win "Git Bash") installed.

Put vupy.sh script in your home folder and add the following line to your `/etc/profile.d/git-prompt.sh` file:

    source /c/Users/<Your User Id Here>/vupy.sh

Then restart your git-bash and you are ready to use vupy.

Run `vupy help` to see all the available options.
